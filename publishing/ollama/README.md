# Ollama Library Submission

## Overview

Submit the CMMC Expert models to the [Ollama Library](https://ollama.com/library) so users can install with:

```bash
ollama pull Nathan-Maine/cmmc-expert-7b
ollama pull Nathan-Maine/cmmc-expert-14b
ollama pull Nathan-Maine/cmmc-expert-32b
ollama pull Nathan-Maine/cmmc-expert-72b
```

## Prerequisites

1. **Ollama account** — Sign up at [ollama.com](https://ollama.com)
2. **GGUF files** — Generated by the training pipeline (`quantized/*.gguf`)
3. **Modelfile** — Already configured at `deployment/Modelfile`

## Steps

### 1. Sign In to Ollama

```bash
ollama login
```

### 2. Create Each Model

From the project root:

```bash
# 7B model
ollama create Nathan-Maine/cmmc-expert-7b -f deployment/Modelfile

# 14B model (update FROM path first)
sed 's|cmmc-expert-7b|cmmc-expert-14b|' deployment/Modelfile | \
  ollama create Nathan-Maine/cmmc-expert-14b -f -

# 32B model
sed 's|cmmc-expert-7b|cmmc-expert-32b|' deployment/Modelfile | \
  ollama create Nathan-Maine/cmmc-expert-32b -f -

# 72B model
sed 's|cmmc-expert-7b|cmmc-expert-72b|' deployment/Modelfile | \
  ollama create Nathan-Maine/cmmc-expert-72b -f -
```

Or use the deployment script which handles this automatically:

```bash
./deployment/setup_ollama.sh 7b
./deployment/setup_ollama.sh 14b
./deployment/setup_ollama.sh 32b
./deployment/setup_ollama.sh 72b
```

### 3. Push to Ollama Library

```bash
ollama push Nathan-Maine/cmmc-expert-7b
ollama push Nathan-Maine/cmmc-expert-14b
ollama push Nathan-Maine/cmmc-expert-32b
ollama push Nathan-Maine/cmmc-expert-72b
```

### 4. Verify

```bash
# Should now be pullable by anyone
ollama pull Nathan-Maine/cmmc-expert-7b
ollama run Nathan-Maine/cmmc-expert-7b "What is CMMC Level 2?"
```
